Como funciona

Simples como apertar um botão

Sem instalar nada extra. Funciona direto no seu navegador, inclusive no celular.

🎙️

1. Abra o app

Acesse app.livetranslator.com.br no seu celular ou computador. Faça login com e-mail — sem configuração de API key.

🔴

2. Aperte gravar

Pressione o botão vermelho e fale (ou apontye o microfone para o áudio). O Web Speech API capta em inglês instantaneamente.

✨

3. Leia a tradução

A IA traduz cada frase para português em menos de 1 segundo. Histórico completo na tela, pronto para copiar ou exportar.

FAQ

Perguntas frequentes

Não! O Live Translator é 100% baseado em navegador. Funciona no Chrome, Edge ou Safari — desktop e celular. Basta criar uma conta e usar. Opcionalmente, existe uma extensão para Chrome que integra diretamente com vídeos do YouTube e Meet.
Cada frase ou trecho de áudio transcrito e traduzido conta como 1 tradução. O contador reseta toda meia-noite (horário de Brasília). Para uso intenso em aulas ou meetings longos, recomendamos o plano Pro com traduções ilimitadas.
O Web Speech API é nativo do Chrome/Edge e funciona offline e com baixa latência — ótimo para uso geral. O Whisper da OpenAI (disponível no plano Pro) é um modelo de transcrição de alta precisão, melhor com sotaques, terminologia técnica e áudio com ruído.
Não armazenamos gravações de áudio. O áudio é processado em tempo real e descartado imediatamente após a transcrição. Apenas as traduções em texto são mantidas temporariamente na sessão do navegador para exibição no histórico.
