FAQ

Perguntas frequentes

Preciso saber programar para usar? +
Não. A instalação é feita diretamente pelo Chrome em 3 cliques. O único passo técnico é criar uma conta gratuita na Anthropic ou OpenAI e copiar a chave de API para o popup da extensão.
Funciona se o professor não ativar as legendas? +
Sim. Neste caso você usa o modo "Áudio do Sistema": a extensão captura tudo que você está ouvindo no fone ou nos alto-falantes e transcreve com Whisper antes de traduzir. A latência é de aproximadamente 5 segundos, mas não depende de nenhuma ação do professor.
Meus dados de áudio ficam armazenados em algum servidor? +
Não passam pelo nosso servidor. O áudio vai diretamente do seu navegador para a API da OpenAI (Whisper) ou Anthropic, usando sua própria chave. Nós não temos acesso a nenhum dado de áudio ou tradução. No plano Pro, os créditos são gerenciados por nós, mas o áudio ainda é processado diretamente nas APIs.
Qual é a qualidade da tradução? +
Muito alta para contexto de aulas e reuniões. O Claude Haiku e o GPT-4o-mini são modelos de linguagem de ponta especialmente bons em tradução contextual. O sistema usa as últimas frases como contexto para resolver pronomes e referências corretamente. O fallback MyMemory é gratuito mas de qualidade inferior.
Funciona em outros idiomas além do inglês? +
A versão atual está otimizada para inglês → português. O Whisper suporta mais de 50 idiomas para transcrição, então uma versão futura poderá oferecer outros pares de idiomas como espanhol, francês ou mandarim → português.
Posso cancelar o plano Pro quando quiser? +
Sim. Sem fidelidade, sem multa. Cancele a qualquer momento pelo painel e você continua com acesso até o fim do período pago. Depois, volta automaticamente para o plano gratuito.
